CONTENTdm is a software tool for organizing, managing, publishing and searching digital collections
Metadata is data about digital objects.
Metadata enables description and discovery (finding) objects, and also enables administration/management of the digital resources.
Many standards, schemas for metadata. 
REALIA metadata scheme is an extension of Dublin Core; REALIA metadata fields mapped to Dublin Core.  Dublin Core is international, extensible standard.
CONTENTdm provides support for the Open Archives Initiative Protocol for Metadata Harvesting Version 2.0 (OAI-PMH v2), an emerging standard for metadata harvesting. CONTENTdm Servers can function as OAI repositories.
