Major criteria for DAM applications
n Support for many object formats, and developing
formats (e.g., jpeg 2000)
n Support for high-resolution, zoom-in features
n Supports Unicode text for display and searching
n URL access to objects
n Customizable display interface
n Based on open standards (database, metadata, etc.)
n Flexible access control list features
n Standards-based export functions to avoid “lock-in” and
promote remote indexing
n Platform (hardware, operating system) agnostic – server
and client
