The Assertions
l OSS is more secure and network-ready
l Usually it is:  e.g., defensively designed Linux
l Users don't have open access to code elements needed to spread
malware
l Proprietary software can be as secure, but fewer developers means
fewer reviews
l OSS can be customized
l OSS is better because it's transparent
l OSS is better because it uses open standards
l So can proprietary software
l OSS is by and for a community
l Experimenting with OSS is expensive
l Not meeting your goals is more expensive
