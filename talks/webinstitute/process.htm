The process of surfing


(1) Connect to the network

(2) Run a web browser

(3) Click on a link

(4) Web browser looks up the "host" of the linked file

(5) Web browser requests that file from the host computer

(6) Host sends the file, or perhaps not

(7) Virtual "connection" to the host is closed

(8) Repeat steps 3 through 7 for hours
