Practical recommendations
n Examine the entire cost
– Licensing, hardware, support, training, documentation, migration
from legacy tools
n Ask why you are considering any application
– Are learning outcomes the driver?
n Pilot the software
– Directly involve all stakeholders; consider outsourcing the pilot
n Start with “low hanging fruit” – not mission critical
applications
n Understand and plan for support needs
n Spend avoided licensing costs on local staff development
n Keep looking – new opportunities arise each week
