The Kenyon Program In Information And Learning Resources
(PILR)
Offerings For Fall Semester, 1998

Workshops | Course-Integrated Instruction | On-Line Tutorials

Kenyon College offers a variety of paths to understanding the best ways to incorporate information technology resources into the academic and administrative challenge. Workshops, on-line tutorials both on the Kenyon network and on the Internet, in-class instruction, documentation, and personal assistance are offered. Below are listed the workshops offered in Fall, 1998, and information on other learning resources presently available..


Workshops

If you have any questions, or would like additional information about a particular workshop or other opportunity, contact the instructor listed for each session or contact PILR (email PILR). Unless otherwise noted below all workshops will be held during Common Hour, 11:50-12:40 on Tuesdays and Thursdays. Prior registration is not required.

Excel: An Introduction
This introductory workshop will cover features of the Excel worksheet; navigating the spreadsheet; entering and editing data; basic formatting commands; saving and printing the spreadsheet; copying and moving data within a spreadsheet; and the File Menu. Instructor: To Be Announced.

Tuesday, October 6 Chalmers Instruction Center

The Internet: Beyond the Basics - Finding Information on the Internet & Customizing Netscape
This workshop will focus on using Internet search engines and subject directories to find information on the Internet. It will also cover basic customization of Netscape for your own computer, including creation and management of bookmarks, etc. Instructor: Jennie Duvernay (duvernayj), x5687.

Thursday, October 15 Chalmers Instruction Center

Introduction To Word Processing at Kenyon
This workshop will introduce beginners to the software available at Kenyon for word processing, with a close examination of the capabilities of WordPerfect. Instructor: Mike McCardel (mccardelm), x5352.

Thursday, September 24 Chalmers Instruction Center
Tuesday, October 13 Chalmers Instruction Center

KCEvents
Learn how KCEvents can help you schedule classrooms and other spaces for campus events. This session is appropriate for faculty, staff, and students. Instructor: Rich Switzer, x5121.

Thursday, September 10 Chalmers Instruction Center

Lexis-Nexis Academic Universe
Lexis-Nexis Academic Universe is now accessible on the web. Academic Universe contains the full text of newspapers and other news sources from around the world. In addition, it provides access to full text federal and state laws and court decisions, as well as scholarly and legal journals. This hands-on session will provide an introduction to techniques for searching. Instructor: Priscilla McIntosh , x5681.

Thursday, September 17 Chalmers Instruction Center
Thursday, October 22 new date Chalmers Instruction Center

Office Software - Tips And Tricks
This workshop is an opportunity for intermediate and advanced users of basic office software products to share their tips and tricks, and learn new ones from others. Some topics in the past have been using graphics in WordPerfect 6.1and 8, and printing in WordPerfect 8. Instructor: Jenny Zolman (zolman), ext. 5696.

Tuesday, October 20 Chalmers Instruction Center

Web Publishing - Introduction to HTML
This introduction to HTML (Hypertext Markup Language) is intended for those who do not know HTML. Lecture and a hands-on exercises creating and testing a simple web page will help the attendee to: understand key acronyms and terms; understand some basic HTML tags and file creation and testing techniques; understand how to create links to other web sites; understand the relationship between HTML and Internet browsers such as Netscape; where to go to get more information on HTML. Instructor: Bill Quimby (quimby), x5120.

Tuesday, September 8 Ascension 01 Lab

Web Publishing - Intermediate HTML
Attendees are expected to have a basic understanding of HTML. Lecture and a hands-on exercise creating a simple web page using the HTML editor Visual Page will help the attendee to: understand how to use an HTML editor to add depth to a home page; understand a few more HTML tags; understand how to add an image to a web page; and understand how to add background color. Instructor: Bill Quimby (quimby), x5120.

Thursday, September 10 Ascension 01 Lab

Web Publishing - Advanced
Attendees are expected to have experience creating simple web pages, and to be planning to place documents or homepages on the Kenyon web server. A hands-on exercise will demonstrate how to move files from a microcomputer to a file server platform using the Windows program WS_FTP. In addition, we will explain and demonstrate advanced HTML features such as tables, forms, frames, Javascript, and Java. Instructor: Bill Quimby (quimby), x5120.

Tuesday, September 22 New Date Ascension 01 Lab

Web Publishing - Introduction To Graphics
This workshop will introduce techniques for the insertion of graphics into an HTML page - beginning with scanning, then going to image editing, and finally coding the image into the document. Prerequisite: You must have attended one or more of the preceding HTML workshops, or obtain permission from the instructor. Instructor: Bill Quimby (quimby), x5120.

Thursday, September 24 New Date Ascension 01 Lab

Web Publishing - Exploring Tables and Frames
This workshop, with exercises, will focus on building two important advanced features into a web page - tables, for the ordered presentation of data (and menus), and frames, which if well done enable the visitor to your web site to more easily navigate through the available material. Prerequisite: You must have attended one or more of the preceding HTML workshops, or obtain permission from the instructor. Instructor: Bill Quimby (quimby), x5120.

Tuesday, September 29 New Date Ascension 01 Lab


Course-Integrated Instruction

The Library and Information Services staff are ready to provide skilled assistance in the classroom. The reference librarians offer many course-related library instruction sessions each semester on basic and advanced research strategies and reference tools relevant to course topics. In addition other LBIS staff offer instruction in computing resources and software applications useful in coursework. Past sessions have included tutorials on web publishing. Faculty interested in course-related computing instruction should contact Bill Quimby, ext. 5120, email QUIMBY. Faculty interested in course-related library instruction should contact their department's library liaison. See the liaison assignments posted on the Kenyon web.
