Kermit Instructions

To START your VAX Session Using a Microcomputer

Start Kermit

Turn on your microcomputer

Type KERMIT R

Log onto VAX

Press R until the username prompt appears.

Type your USERNAME R

Type C ACAD R (for students), or C ADMIN R (for staff and faculty), or C LIBRARY R (for Library use)

Type your USERNAME and press R

Type your PASSWORD and press R

Wait for the VAX $ prompt. At the $ prompt, you may start Mail or WordPerfect or other VAX programs OR transfer files.


To Transfer Files

Move to Directory Where Your Files Are

Type SET DEFAULT [.directoryname] R

For example,

SET DEFAULT [.WP] R

Start Kermit on the VAX

At the $ prompt type KERMIT R

Change to Server Mode

At the C-Kermit> prompt type Server R

Switch Back to the Micro

Press A C

Set Micro Default Drive

At the MS-Kermit> prompt type SET DEF A: OR SET DEF B: or SET DEF C:\directoryname as the case may be, and press R .

"Get" or "Send" your Files

Type GET filename R if you are getting a file FROM the VAX.

Type SEND filename R if you are sending a file TO the VAX.

When you are finished

At the MS-Kermit> prompt type FINISH R

Connect back to VAX

Type Connect or C R


To Stop Your VAX Session Using a Microcomputer

Log off VAX

At the $ prompt type LO R

Log off server

At the LOCAL> prompt type LO R

Connect back to microcomputer

Press A C

End Kermit session

At the MS-KERMIT> prompt type EXIT R
